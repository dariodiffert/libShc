First steps

The libShc has been designed to be as easy to use as possible while maintaining a high degree of flexibility.

The class Shx uses the external libs OpenCV (Installation) to read and write panoramic images. Furthermore OCamCalib (Installation) calibration files can be read to automatically unwrap images from panoramic sensors. See libShc/src_examples/simple.cpp for a basic example of how to use the libShc.

The libShc provides a wide range of functions to work with Fourier Transformed data (on the sphere), the most important are described in Spherical Harmonics & Fourier Transform, Surface, Visual Compass, Rotations, Translations (Warping), Products, Spectra, and Pointclouds.

To get an idea of how to work with the libShc, you should read the basics about the Class Layout, Coordinate System, Storing Orders, Hemispherical Continuations, Rotation Parameters and File Input. Various examples are provided in the folder libShc/src_examples.

Finally, make sure to read the Pitfalls section carefully, since you might encounter these while using libShc.

Advanced:

While panoramic images provide the easiest way to handle functions defined on the unit sphere, it is also possible to directly access the Surface. This can be used to work with arbitrary surface data or to decrease the overhead of the image/matrix pipeline. The latter is important if the application is time critical. The Tangent Distance can be used to increase the invariance against arbitrary transformations.
