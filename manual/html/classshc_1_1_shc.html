void shc::Shc::init_tolerances ( float  translation,
float  cg 
)

Initializes the number of bands (frequencies) used for computations in the basis of spherical harmonics.

Translation and Clebsch-Gordan matrices (Translations (Warping), Products) can contain many values close to zero. By setting the corresponding tolerance value, entries whose absolute values are small can also be set to zero. Due to the increased sparsity of the matrices, this can reduce the computation times. Note that by setting translation greater zero, translation matrices will be represented by sparse matrices instead of dense matrices. This might actually increase the computation times if the tolerance value is too small.

Initialization: This function is required to be called before the instance of the shc::Shc class is initialized (Shc::init). If not used, a default set of parameters is used (see console output during initialization).

Parameters
[in]translationThreshold for translation matrices
[in]cgThreshold for Clebsch-Gordan matrices

Initializes the file names in which precomputed values are saved.

Surfaces and translation information (Surface, Translations (Warping)) are precomputed. To avoid the computation of these information at each initialization, these information can be stored in files. After successfull initialization, the information are automatically stored in the specified files. If the files exist at initialization, the information are loaded from the specified files. If the loaded information differs from the used initialization settings, a warning is shown.

Initialization: This function is required to be called before the instance of the shc::Shc class is initialized (Shc::init). If not used, a default set of parameters is used (see console output during initialization).

Parameters

[in]file_translationName of the file where the translation matrices are stored.
[in]force_overwriteIf the loaded information differs from the used initialization settings, the initialization is redone with the current settings and afterwards saved to the specified files. This setting avoids using accidentally old settings if these differ from the currently used settings.

Initializes the number of bands (frequencies) used for computations.

Sets the maximal number of bands used for computations in the basis of spherical harmonics. This setting affects all computations in the basis of spherical harmonics, e.g. the Fourier transform, rotations, translations, etc. Most functions have an additional parameter to use a smaller number of bands, however it is not possible to increase the number of bands after initialization.

Parameters

Initializes the surface used for the Fourier transform (grid)

To compute the (discrete) Fourier transform, a set of sampling points has to be specified (Surface). This function creates a grid of width x height sampling points for the spherical coordinates.
