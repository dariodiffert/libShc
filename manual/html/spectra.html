Spectra

The amplitude spectrum and bispectrum can be calculated directly from any shc::Shpm instance of Fourier transformed data via shc::Shc::get_feature (note the Storing Orders for both spectra). Both spectra are rotational-invariant and can be used as a distance measure between two functions by shc::Shc::get_feature_difference.

Note that the amplitude spectrum does only contain the amplitude information of a given function. The bispectrum contains both amplitude and phase information.

Since Clebsch-Gordan coefficients (see Products) are needed in order to calculate the bispectrum, the number of bands for which the Clebsch-Gordan coefficients are calculated have to be initalized first via shc::Shc::init_bands.

The bispectrum is strongly related to Products and is calculated as described in Kakarala and D. Mao (2010) (see Literature).
