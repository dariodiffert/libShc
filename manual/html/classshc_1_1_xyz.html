Detailed Description

Representation of a rotation via the X"Y'Z Tait-Bryan convention.

This class is used to represent rotations via the X"Y'Z Tait-Bryan convention, that is a rotation around the Z-axis, followed by a rotation around the Y-axis, and finally a rotation around the X-axis. Make sure to use the correct Coordinate System.
